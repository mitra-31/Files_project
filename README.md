Usage:

    main.py add <item> <price> <catogery>

    main.py modify <case> <item> [price] <catogery>

    main.py menu (--a|--r <catogery>)


Helper:

    <catogery> = 's' : 'STARTERS',
  
                'b' : "BIRIYANIS",
  
                't' : "THALIS"

Options:

    -l,--list  list all.

    -h,--help  show commands.
